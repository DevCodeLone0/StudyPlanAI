# StudyPlanAI

> AI-Powered Study Companion with Gamification

## 🎯 Overview

StudyPlanAI is a web application that helps students create personalized study plans powered by AI. The AI acts as a virtual tutor, generating curriculum, monitoring progress, and adapting plans based on performance. A gamification layer keeps students motivated through XP, streaks, badges, and levels.

## ✨ Features

### 📚 AI-Powered Plan Generation
- Input your study goals and receive a complete curriculum
- Plans organized into modules with milestones
- AI suggests adjustments based on your pace

### 🎮 Gamification
- Earn XP for completing milestones
- Build daily study streaks
- Unlock badges and achievements
- Level up as you progress

### 🤖 AI Tutor
- Real-time chat with AI assistant
- Weekly progress reports
- Dynamic plan adaptation
- Motivational support

### 📊 Dashboard
- Visual progress tracking
- Activity calendar (streak heatmap)
- Achievement showcase
- Next milestone preview

### 👨‍💼 Admin Panel
- User management
- AI configuration
- System analytics
